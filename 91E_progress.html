What is programming?

Check out picture samples at the end of the article

Programming is the process of writing lines of codes using various programming languages depending on the software being developed. These programming languages have evolved overtime, fortunately, subsequently humans have been able to develop more complex systems using more sophisticated languages to suite the fast-rising technological era significantly marked by invention of AI(Artificial Intelligence) to mimic the cognitive reasoning capability of human beings. It has noticeably been marked via Robotics, Natural Language Processing (NLP) and Expert systems.

Evolution of Programming languages

Types of programming languages

  1. Machine Language
  2. Assembly language
  3. Structured Programming language
  4. 4GL
  5. 5GL

Since the inception of time,developments emerged from Machine language in the first generation characterised by using binary, making it machine oriented(could not be used in more than one computer) as well as less user-friendly, onto assembly language which used mnemonic codes during the second generation, then onto the Structured programming languages which made a remarkable renovation from its predecessors by introducing modules and control structures in its algorithm. However, HTML( Hypertext Markup Language) is not considered as such because it has no variable declaration neither does it have control structure therefore, it is considered as a markup language - as its name suggests(lol >=<). Consequently as time and years went by, humans invented new languages such as 4GLs which were more user-friendly as they were easily used since they used "menus"; a good and appropriate example is Visual basic.
